## Rules
1. One active project per working block. Not per day — per block.
2. New idea mid-work → `IDEAS.md` at repo root. Do not open a new project file.
3. Update "Next action" on a project *before* you stop working on it.
4. Weekly review (pick one day): clear `IDEAS.md`, promote anything worth doing into a real project.
